# AI Resume ATS Analyzer 🚀

An AI-powered application designed to streamline the mock interview and resume optimization process. This tool helps job seekers improve their resumes and interview performance by leveraging advanced Large Language Models (LLMs).

## 💡 Key Features
* **ATS Analysis:** Evaluate how well a resume aligns with specific job descriptions.
* **Skill Extraction:** Automatically identify and highlight key technical skills from resumes.
* **Mock Interviews:** Generate role-specific interview questions based on the candidate's profile.
* **Resume Rewriting:** Provide suggestions to enhance resume content for better impact.

## 🛠 Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **AI Engine:** `Groq API` 
* **LLM Integration:** LangChain
